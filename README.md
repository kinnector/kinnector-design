# kinnector-design

UI/UX specifications, design assets, style guides, and design novel specifications for the Kinnector platforms.

## Key Files & Assets

- **[DESIGN-SYSTEM.md](file:///home/user/Documents/kinnector/kinnector-design/DESIGN-SYSTEM.md)**: Visual identity documentation including color tokens (HSL, dark/light modes, gradients), typography specifications (Outfit, Inter), layouts, interactive states, and micro-animation specs.
- **[DESIGN-NOVEL.md](file:///home/user/Documents/kinnector/kinnector-design/DESIGN-NOVEL.md)**: Conceptual blueprints, immersive UI walkthroughs, and story-driven user flow definitions.

## Design Principles

1. **Vibrant Visuals**: Curated dark modes, modern typography, glassmorphism, and smooth gradient backgrounds.
2. **Interactive States**: Elevate interactions using subtle hover transitions and micro-animations.
3. **Consistency**: Follow tokenized variables defined in the design system for CSS, web apps (Svelte 5/Vite 6/Tailwind v4), and native platforms.
